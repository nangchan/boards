# Nangchan

## Leaderboard

Sample project to display won/open leaders

- React (with Hooks)
- Material-UI
- Recharts

http://localhost:4200/

![screenshot](apps/leaderboard/src/images/screenshot.png)

### To clone

    git clone https://github.com/nangchan/boards.git

### To install

    cd boards
    npm install

### To run the app

    npm start

### To run the tests

    npm test

## To generate dependency graph

    npm run dep-graph
